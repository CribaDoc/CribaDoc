Necesario: dockerdesktop instalado y abierto 


1."Instalar" CribaDoc:

-DockerDesktop Abierto
-Abrir terminal en carpeta del repo -> docker compose up -d --build

2. Arrancar/Detener ejecución

-docker compose stop
-docker compose start

3.Borrar Datos CribaDock

-docker compose down -v
*Pierdes los datos con este ultimo
